# Search-Firewall-Extension
Block unwanted domains in search results on Helium/Chrome/etc.

Clean up search engine results, Works on Google, Startpage, DuckDuckGo…

Just bare minimum nice working extension to elliminate irrellevant domains from search results.

Example: Seearch > Build an engine > elliminate youtube and facebook search results by inserting youtube.com facebook.com in the expluded domains (manual edit)

Comes with other features, extension bar icon enable that in the extension details, click to quick add or remove domains.

Search page UI has buttons to block domains quickly.

Manifest V3 extension.
